# Safety Checks

## Before editing code

- [ ] Read context
- [ ] Confirm task scope
- [ ] Identify exact files
- [ ] Check tier impact
- [ ] Check WebView2 impact
- [ ] Check release/update impact
- [ ] Check config/schema impact

## Never do silently

- [ ] Rename controls
- [ ] Move WinForms files
- [ ] Change Designer files
- [ ] Change release scripts
- [ ] Change server.js
- [ ] Change Firebase schema
- [ ] Change Supabase manifest schema
- [ ] Log full tokens
- [ ] Enable BASE background sync

## Stop and ask if

- Task requires architecture refactor
- Task touches MainForm large sections
- Task touches FullStackOperation large sections
- Task changes release pipeline
- Task changes license/security
- Task changes WebView2 automation selectors
